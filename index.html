About

Games captured my imagination from an early age—how rules, visuals, and sound could intertwine to create something alive. These days, I like to craft small interactive experiences where players’ choices leave a mark, and where mechanics can surprise both the player and myself.

Since I started exploring game development in 2020, I’ve been learning by doing—tinkering with AI, testing dynamic gameplay ideas, and shaping worlds that didn’t exist before. Not every experiment worked, but each taught me something new, and the worlds that did come to life keep fueling my curiosity to push further.

Projects

Rhythm Game (Unity)

A timing-based rhythm game focused on input accuracy and synchronization with music. The project experiments with hit windows, feedback systems, and precise input handling to make each note feel satisfying. Through this, I learned how to balance difficulty, polish moment-to-moment gameplay, and design systems where even small mistakes feel intentional.

MusiCLI (Python)

A terminal-based music recommendation system that suggests tracks based on mood and text input. The project uses simple NLP techniques and rule-based logic to interpret user intent and generate recommendations. It was an experiment in combining programming with personal interests, focusing more on logic and interaction than flashy visuals.

Voice Assistant for PWD (Python)

An offline, terminal-driven voice assistant for people with disabilities designed for basic tasks like reminders, notes, and command execution. This project focuses on privacy-first design and local processing rather than cloud-based services. It helped me understand speech input pipelines, command parsing, and building usable tools with minimal interfaces.

First-Person Movement System (Unity)

A custom first-person movement system built in Unity with a focus on responsiveness and player feel. This project explores grounded movement mechanics like acceleration, air control, jumping, and camera handling. I used it as a sandbox to understand how small changes in physics and input timing affect gameplay, and as a foundation for future action-focused games.
